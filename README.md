# flora-vision


# Project Submission

Flora Vision is a modern and responsive plant e-commerce frontend built using React, Vite, and Tailwind CSS. The project focuses on creating a clean and user-friendly interface with reusable components and responsive layouts.


# Project Features.

Responsive navigation bar.
Modern hero section.
Trending products section.
Top-selling products section.
Customer review section.
Reusable React components.
Fully responsive design for all devices.

# Tech Stack

React.js
Vite
Tailwind CSS
JavaScript

# Setup & Run Instructions.

1. Clone the Repository
git clone https://github.com/your-username/flora-vision.git
2. Move to Project Folder
cd flora-vision
3. Install Dependencies
npm install
4. Run the Development Server
npm run dev

The application will run locally on:

http://localhost:5173
